# Эта программа:
1. Находит группы орфографически похожих слов.
2. Определяет, сколько раз слова из каждой группы встречаются в тексте рядом с другими словами из той же группы.
3. Выводит результат в заданном формате.

# Как работать с этой программой?
### Формат ввода
* Первая строка: целое число K (1 ≤ K ≤ 10)
* Далее — строки текста, состоящие из слов, разделённых " ".
* Текст заканчивается пустой строкой.
* Слова содержат латинские буквы, апострофы и знаки препинания.
Гарантируется: общее количество слов ≤ 100 000.
### Формат вывода
* Для каждой группы, у которой контекстная частота > 0:
* Выводит: <представитель>: <частота>
* Представитель — лексикографически наименьшее слово в группе.
* Группы выводит:
  - По убыванию частоты
  - При равенстве — по алфавиту представителя.
